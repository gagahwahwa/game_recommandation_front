# Sourgrape

This project is game recommand web application.

### Using Tech
- **Front-end**
  - Angular 4.x
    - Rxjs
    - Typescript
  - SCSS
  - PWA (*may be*)
  
- **Back-end**
  - Nodejs + Express
  - Mysql
  - Hadoop (*may be*)
  - Elastic Search (*may be*)
  
### Using Algo
- CF ( Collaboration Filtering )
  - It compute predicting rate.
- CBF ( Context Based Filtering )
  - It filter user interest.
